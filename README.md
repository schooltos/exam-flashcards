# Flashcards App

Мобільний додаток для навчання за допомогою карток, розроблений на React Native з використанням Expo.

Проєкт виконано для залікової роботи з дисципліни **«Розробка інтерфейсів користувача»**.

## Варіант завдання

**Варіант 32.** Реалізувати додаток для навчання з картками Flashcards: створення колод, додавання карток із питанням та відповіддю, режим вивчення з перевертанням картки по тапу.

## Опис проєкту

Додаток дозволяє створювати навчальні колоди та додавати до них картки у форматі «питання — відповідь». Користувач може переглядати створені колоди, відкривати окрему колоду, додавати нові картки та запускати режим навчання. У режимі навчання картка перевертається після натискання, що дозволяє спочатку побачити питання, а потім перевірити відповідь.

Дані зберігаються локально на пристрої за допомогою AsyncStorage, тому створені колоди й картки залишаються доступними після перезапуску додатку.

## Основні можливості

* перегляд списку навчальних колод;
* створення нової колоди;
* відкриття окремої колоди;
* додавання карток із питанням та відповіддю;
* перегляд списку карток у колоді;
* режим навчання;
* перевертання картки по тапу;
* перехід між картками у режимі навчання;
* локальне збереження даних;
* завантаження демонстраційних даних для швидкої перевірки роботи додатку.

## Встановлення та запуск

Для запуску проєкту потрібно встановити Node.js та npm.

### 1. Клонування репозиторію

```bash
git clone https://github.com/schooltos/exam-flashcards.git
```

### 2. Перехід у папку проєкту

```bash
cd exam-flashcards
```

### 3. Встановлення залежностей

```bash
npm install
```

### 4. Запуск Expo-сервера

```bash
npm start
```

Після запуску в терміналі з’явиться QR-код. Його потрібно відсканувати через додаток **Expo Go** на телефоні.

Проєкт тестувався через **Expo Go 54.0.2**.

## Запуск на мобільному пристрої

Для запуску на телефоні потрібно:

1. Встановити додаток **Expo Go**.
2. Підключити телефон і комп’ютер до однієї Wi-Fi мережі.
3. Запустити проєкт командою:

```bash
npm start
```

4. Відсканувати QR-код через Expo Go.
5. Дочекатися завантаження мобільного додатку.

## Сценарій використання

Після запуску додатку користувач бачить список колод. Якщо колод ще немає, можна створити нову колоду або завантажити демонстраційні дані. Після створення колоди користувач відкриває її та додає картки. Кожна картка містить питання і відповідь.

У режимі навчання користувач бачить питання на картці. Після натискання на картку вона перевертається і показує відповідь. За допомогою кнопок переходу можна переглядати наступні та попередні картки.

## Локальне збереження даних

Для збереження колод і карток використовується AsyncStorage. Це дозволяє не втрачати дані після закриття додатку. Усі створені користувачем колоди та картки зберігаються локально на пристрої.

## Демо-дані

У додатку передбачено кнопку для завантаження демонстраційних даних. Вона потрібна для швидкої перевірки основного функціоналу без ручного створення колод і карток.

Демо-дані дозволяють одразу протестувати:

* перегляд колод;
* відкриття колоди;
* перегляд карток;
* режим навчання;
* перевертання картки по тапу.
